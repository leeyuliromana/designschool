웹 개발 기초

웹 개발에 필요한 프로그램

  • web browser
  • visual studio code
  • git

카레를 만드는 방법

  1. 야채를 깍뚝썰기 해서 준비한다.
  2. 물을 풀고 카레 가루를 넣는다.
  3. 카레가 끓으면 썰어놓은 야채를 넣는다.
  4. 완성되면 밥과 함께 맛있게 냠냠.

웹관련 용어 정리

html
hyper text mark up language 의 약자로 웹사이트의 구조를 설계할때 사용하는 언어 이다.
css
cascading style sheet의 약자로 마트업 요소에 스타일을 정의할 때 사용하는 언어이다.
